Design exploration · docs/mockups/07

Clarity — the brass plaque, redrawn for a phone.

The tradition this draws from: the brass plaque by the door and the superintendent's log — the clarity of well-set print, not a SaaS dashboard.

  1. 1Competitors are dashboards; Stoop is a feed — one decision, one screen, one button, not a control panel to scan.
  2. 2Reasoning is trust, so it reads like an editor's margin note next to the reply — a plain sentence, visible by default, never tucked behind a disclosure toggle.
  3. 3Severity is a plain word set like the enamel plaque by the door — Emergency, Can't wait, Whenever — read in a glance, never a coded tag.
  4. 4Type and touch sized for a landlord on a job site, not a designer's laptop: 16px+ body, 48px+ targets, one composed serif voice for Stoop, one plain voice for the tenant's actual texts.
  5. 5No separate desktop product — the same feed just gets more margin to breathe.

Do

  • 16px+ type, 48px+ targets
  • One action per screen
  • Reasoning written in plain view
  • Severity in plain words
  • Four tabs, no more

Don't

  • Rounded-card dashboards
  • Icon-only buttons
  • Purple gradients, glassmorphism
  • "Triage," "AI agent," jargon
  • Sidebars or breadcrumbs
